# Chapter 5 - High-Level System Architecture

Parker is organised into six layers: Experience, Cognitive, Trust, Runtime, Integration and Infrastructure.

Experience interacts. Cognition reasons. Trust authorises. Runtime executes. Integration connects. Infrastructure supports.

See the [Parker Constitution](parker-constitution.md) for the constitutional form of this layering: cognition proposes, trust authorises, runtime executes, and no layer may absorb another.
