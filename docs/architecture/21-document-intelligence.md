# Chapter 21 – Document Intelligence

## Purpose

The Document Intelligence Service enables Parker to understand, classify, extract, summarise and reason over user documents while preserving privacy and user control.

## Responsibilities

- Document ingestion
- OCR
- Layout analysis
- Classification
- Entity extraction
- Semantic indexing
- Confidence scoring
- Structured export

## Architectural Rules

- Original documents remain immutable.
- Extracted knowledge remains traceable to source.
- Confidence accompanies every extracted fact.
