# Chapter 31 – Search Service

## Purpose
Provides a unified search capability across every Parker resource.

## Responsibilities
- Search Memory
- Search World Model
- Search Documents
- Search Email
- Search Calendar
- Search Contacts
- Search Home Assistant
- Rank results
- Apply permission filtering
- Return citations

## Rules
- Search never bypasses permissions.
- Results remain traceable.
- Confidence accompanies every result.
